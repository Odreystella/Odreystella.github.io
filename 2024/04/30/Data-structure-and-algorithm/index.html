자료구조

  • 개발자가 데이터를 효율적으로 사용할 수 있도록 정리하는 방법
  • 데이터를 체계적으로 저장하고, 효율적으로 활용하기 위함
  • 각각의 자료구조에는 장단점이 있기 때문에 해결하고자 하는 문제에 어떤 자료구조가 맞는지 우선적으로 판단해야함

자료구조의 분류

알고리즘

  • 문제를 해결하기 위해 필요한 계산 절차나 처리 과정의 순서를 나열한 것

리스트 연산

  • 원소 덧붙이기 .append() O(1)
  • 끝에서 꺼내기 .pop() O(1)
  • 원소 삽입하기 .insert() O(N)
  • 원소 삭제하기 .remove() O(N)
  • 원소 인덱스 가져오기 .index() O(N)

탐색 알고리즘

  • 선형 탐색 O(N)
  • 이진 탐색
    • 탐색하려는 리스트가 이미 정렬되어 있는 경우에만 사용 가능
    • 한번 비교가 일어난 때마다 리스트의 길이가 반씩 줄어듦
    • O(logN)
