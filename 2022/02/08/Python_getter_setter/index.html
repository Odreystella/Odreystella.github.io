Getter, Setter
  • 인스턴스의 내부에서 데이터를 안전하게 접근하기 위해 사용하는 키워드이고, 객체지향 프로그래밍에서 흔히 볼 수 있는 패턴이다.
    • Getter: 데이터를 읽어주는 메서드
    • Setter: 데이터를 변경해주는 메서드
  • 클래스 안의 인스턴스 변수를 직접 접근하여 값을 가져오거나 수정하지 않고, 이것을 해주는 메서드를 만들어 메서드를 통해서 가져오는 것을 권장함.
  • 메서드 이름은 변수명과 동일하게 작성하는 관례가 있다.
  • 파이썬의 내장함수는 property()메서드를 사용하면 깔끔하게 getter, setter 메서드를 호출 할 수 있고, 내장 데코레이터인 @property 를 활용할 수 있다.
Getter, Setter 구현 방법
  1. property() 함수를 이용하는 방법
  2. @property 속성을 이용하는 방법
