PEP 8 -> Python Enhance Proposal(파이썬 개선 제안서)

  • 들여쓰기(Indentation)
    • 공백 4칸 권장
    • cf. 파이썬에서는 space와 tab를 혼용해서 쓰지 못함
  • 한 줄에 최대 79자까지
  • 최상위 함수와 클래스 정의는 2줄씩 띄워 사용하기
  • import는 항상 소스코드 맨 위에 써야하면 분리해서 작성하기
    • Good) import sys
    • Good) import os
    • Bad) import sys, os

DB 모델링

  • PrimaryKey(기본키, pk)
    • 테이블에 저장된 각각의 데이터를 유일하게 구분하는 키
    • 즉, 데이터베이스 테이블에 각 행의 식별 기준인 기본키가 필요함
    • Django에는 기본키로 id가 디폴트로 지정되어 있다.
    • id = models.AutoField(primary_key=True)
  • ForeignKey(외래키, fk)
    • 각 테이블 간에 연결을 위해서 특정 테이블에서 다른 테이블을 참조하는 기본키를 말함

Django FieldType

모든 필드타입 클래스들은 추상 클래스 Field 클래스의 파생 클래스

  • CharField(max_length=64)
  • TextField
  • IntegerField
  • FloatField
  • BooleanField
  • DateTimeField
  • FileField
  • EmailField(Email Validator 사용)
  • ImageField(FileField의 파생클래스)
  • DecimalField
  • AutoField
  • DateField(auto_now=False, auto_now_add=False, **options) - auto_now : 데이터가 저장될 때(update), auto_now_add(create)
