6차 과제의 요구사항을 간략하게 정리해보자

과제의 기한과 요구사항은 다음과 같았다.

  • 기한: 21/11/8 16:00 ~ 11/10 10:00 (약 1.5일)
  • 개발 요구사항:
    • 선호 기술 스택: flask or fastapi
    • DB: RDBMS 사용
    • REST API 기능
      • 회사명 자동 완성
        • 회사명 일부만 들어가도 검색이 되어야 함
      • 회사 이름으로 회사 검색
      • 새로운 회사 추가
    • 제공되는 test case를 통과할 수 있도록 개발해야 함
    • ORM 사용
    • 결과는 JSON 형태
    • Docker 사용 시 가산점 있음
